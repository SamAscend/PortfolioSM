# Samuel M. Sinurat | Portfolio

> "Every masterpiece begins with a single decision."

## 🚀 About This Portfolio

Personal portfolio website built with pure **HTML, CSS, JavaScript**, and **Three.js** for 3D particle background. Showcases my journey as a Frontend Developer, AI Engineer, and Computer Science student.

## ✨ Features

- 🎨 **Monochrome Theme** with Cyan/Teal accents
- 🌟 **3D Particle Background** using Three.js
- ⌨️ **Typing Effect** for dynamic titles
- 📱 **Fully Responsive** (Mobile-first)
- 🌙 **Dark Mode Toggle**
- 🌀 **Parallax Scrolling**
- 📊 **Timeline Experience** with hover effects
- 🏆 **Project Cards** showcasing my best work

## 🛠️ Tech Stack

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Vanilla)
- Three.js (3D Graphics)
- Font Awesome (Icons)
